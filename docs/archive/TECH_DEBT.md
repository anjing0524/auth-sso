## 债务管理流程

1. **发现** → 代码审查 / 自动化审计 / 安全演练
2. **记录** → 更新本文件 + 关联 CAR 文档
3. **排期** → 每迭代 planning 时 review 本清单
4. **清理** → 修复后标记完成日期 + 更新 CAR 效果度量
5. **回顾** → 季度回顾债务趋势（总量、平均 age、P0 占比）
